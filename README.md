## Design System


### Sobre

> Projeto de um design system, onde foram utilizados principalmente o React e o Storybook para criação do mesmo. Além disso bibliotecas como TypeScript, Stitches e Radix foram utilizados para compor o design system. Projeto muito enriquecedor, principalmente na parte do storybook, onde vemos a importância da padronização de estilos.



### Tecnologia usadas com suas versões

- Reactjs - 18.3.1
- Typescript - 5.4.5
- tsup - 8.0.2
- Phosphor-react - 1.4.1
- Stitches - 1.2.8
- Eslint -  8.57.0
- Storybook - 8.1.9
- Vite - 5.2.0

### Execução do projeto

**Requer o NODE (npm) e o GIT instalado!**
~~~javascript
 1. git clone https://github.com/wwilliamsantana/design-system-ig.git
 2. cd .\design-system-ig\
 3. npm install
 4. npm run dev 
~~~~
